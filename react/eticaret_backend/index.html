Ladies REST API Ana Sayfası

Ladies REST API ana sayfasına hoşgeldiniz! Bu API aracılığıyla Ladies sitesinden belli özelliklere göre ürün çekme işlemlerini yapabilirsiniz.

  • /get: Sitede bulunan bütün ürünleri getirir
  • /get/category: Sitede bulunan belli kategorideki ürünleri getirir
    • Kategori adı; Makyaj, Parfüm, Cilt Bakışı(typo), Vücüt ve Banyo, veya Saç olabilir
  • /get/brand: Sitede bulunan belli markanın ürettiği ürünleri getirir
  • /get/routes: Sitede bulunan ürünlerin özel sayfalarının rotalarını döndürür
